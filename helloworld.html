create a dev folder
    
      mkdir -p dev
      cd dev
      mkdir helloworld && cd helloworld
    

  

  
create some files
    
      touch hello_world.py
      touch hello_world.rb
      touch hello_world.js
    

  

  
open the text editor
    
      atom .
    

  

  
python

  
in hello_world.py:
    
      print("Hello, World")
    

  

  
in the terminal:
    
      python hello_world.py
    

  

  

  
ruby

  
in hello_world.rb:
    
      puts "Hello, World"
    

  

  
in the terminal:
    
      ruby hello_world.rb
    

  

  

  
node

  
in hello_world.js:
    
      console.log("Hello, World");
    

  

  
in the terminal:
    
      node hello_world.js
